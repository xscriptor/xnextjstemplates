# X Editorial · Template (EN/ES)

[README en español](./README.md)

Next.js (App Router) template for a literary publishing house: sober visuals, classic typography and a complete page structure.

## Features

- EB Garamond typography
- Light/dark mode toggle in the top navigation
- Spanish/English variant with an ESP/ENG toggle
- Responsive navigation (desktop + mobile menu)
- Pages: home, catalog, authors, submissions, about, contact and legal

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Routes
