# X Editorial · Plantilla (ES/EN)

[README in English](./README.en.md)

Plantilla de Next.js (App Router) para una editorial literaria con estética sobria, tipografía clásica y estructura completa de páginas.

## Características

- Tipografía EB Garamond
- Modo claro/oscuro con selector en la barra de navegación
- Variante en español e inglés con selector ESP/ENG
- Navegación responsive (desktop + menú móvil)
- Páginas: inicio, catálogo, autores, convocatorias, sobre la editorial, contacto y legal

## Desarrollo

```bash
npm install
npm run dev
```

Abrir http://localhost:3000
