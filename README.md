# Xscriptor Next.js Templates Collection

Este es un repositorio de colección de plantillas y elementos para Next.js en TypeScript con CSS y Tailwind, diseñado para facilitar y acelerar el desarrollo de sitios web modernos.

[Read in English](./README.en.md)

## Contenido del Repositorio

Aquí encontrarás las siguientes plantillas y proyectos:

- **Business**: Plantilla para sitios corporativos y de negocios.
  - [Ver documentación](./business/README.md)

- **Editorial**: Plantilla enfocada en contenido editorial, blogs y publicaciones.
  - [Ver documentación](./editorial/README.md)

- **Music**: Plantilla para artistas, bandas y sitios relacionados con la música.
  - [Ver documentación](./music/README.md)

Cada carpeta contiene su propio `README.md` con instrucciones específicas de instalación, configuración y uso.
